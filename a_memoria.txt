MEMORIA - PRÁCTICA DE LABORATORIO 2
PROCESADORES DEL LENGUAJE
=====================================

DESCRIPCIÓN DEL PROYECTO
-------------------------
Este proyecto contiene los archivos necesarios para la práctica de laboratorio 2 
de Procesadores del Lenguaje.

ARCHIVOS INCLUIDOS
------------------
1. archivo_csv.csv
   - Archivo CSV con datos de ejemplo
   - Contiene información de personas (nombre, apellido, edad, ciudad)
   - Formato: CSV separado por comas

2. Main.java
   - Programa Java para leer archivos CSV
   - Lee el archivo archivo_csv.csv
   - Muestra los datos en consola de forma organizada
   - Incluye manejo de excepciones

3. memoria.txt
   - Este documento
   - Documentación del proyecto

INSTRUCCIONES DE USO
---------------------
Para compilar y ejecutar el programa Java:

1. Compilar:
   javac Main.java

2. Ejecutar:
   java Main

REQUISITOS
----------
- Java JDK 8 o superior
- El archivo archivo_csv.csv debe estar en el mismo directorio que Main.java

FUNCIONAMIENTO
--------------
El programa Main.java:
- Abre y lee el archivo CSV línea por línea
- Identifica los encabezados de las columnas
- Procesa cada registro y muestra los datos
- Maneja errores de lectura de archivo
- Muestra un resumen del total de registros leídos
